What is DOM? Explain purpose of DOM!

DOM is a Document Object Model & Programming Interface for HTML that defines objects, properties, methods and events. The purpose of DOM is to get, change, add, delete HTML elements.

How can you select any element by DOM?
HTML elements mainly select by four ways. Those are:
  1. Selecting Elements by ID: The selection is based on the Id name of HTML element.
  2. Selecting Elements by Class Name: All the HTML elements that matches the specified Class name.
  3. Selecting Elements by Tag Name: All the HTML elements that matches the specified Tag name.
  4. Selecting Elements with CSS Selectors:
    • Query Selector: First element of HTML that matches a specified CSS selector (id, class names, types, attributes, values of attributes, etc).
    • Query Selector All: All the HTML elements that matches a specified CSS selector (id, class names, types, attributes, values of attributes, etc).
What is Event Bubble?

When we fire any event then the inner most element's (target element) event is handled first and then the outer (parent and ancestor elements in the DOM tree until it gets to the root element). This method called Event Bubble.

What is Callback Function?

A callback function is a function that is passed as an argument to another function. It allows a function to call another function and also can run after another function has finished.
